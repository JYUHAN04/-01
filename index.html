完整中文使用教程

① 修改网站访问密码操作步骤

  1. 打开底部「设置」页，找到「访问密码」。
  2. 输入当前密码，再输入两遍新密码。
  3. 点击「保存新密码」，页面会自动保存并生成临时本地备份。
  4. 首次使用的默认密码是 5201314，建议两个人确认后尽快修改。

② 在线编辑内容、上传图片操作方法

  1. 首页昵称、生日、身高、爱好、城市、恋爱标语等输入后会自动保存。
  2. 记录板块每个模块都可以点击「新增」添加内容，已有内容可点「编辑」或「删除」。
  3. 头像、记录配图、相册图片都通过网页上传，图片会压缩后转成 Base64 存在本机浏览器里。
  4. 相册支持一次选择多张图片；单张图片可添加备注、删除，也可点开进入全屏轮播。

③ 数据包导入导出、两人数据同步完整流程

  1. 一方点击页面固定的「一键备份」或设置页「导出全部数据」，得到 JSON 数据包。
  2. 通过微信把 JSON 文件发送给另一半。
  3. 另一半打开网站，进入「设置」，点击「导入数据包」并选择收到的 JSON 文件。
  4. 导入会替换当前本机数据，完成后所有清单、日记、相册 Base64 图片和设置都会同步。
  5. 备份文件包含私密图文，请不要转发给外人，也不要上传到陌生网盘。

④ PWA 安装到手机桌面操作指南

  1. 安卓 Chrome、Edge 等浏览器通常可在菜单里选择「添加到主屏幕」或「安装应用」。
  2. 苹果手机仅 Safari 支持添加到主屏幕：用 Safari 打开后，点分享按钮，再选「添加到主屏幕」。
  3. 微信内置浏览器不支持完整 PWA 安装，请先用系统浏览器打开。
  4. 若直接打开本地 HTML 时浏览器限制 PWA，数据功能仍可正常离线使用；需要安装体验时可用手机浏览器支持的本地页面方式打开。

⑤ 使用重要提醒

  1. 尽量避免长期使用微信内置浏览器打开，文件选择、导入导出和安装到桌面可能异常。
  2. LocalStorage 属于浏览器本地缓存，清理浏览器数据、换手机或卸载浏览器都可能丢失。
  3. 养成定期点击「一键备份」导出数据包的习惯，尤其是上传相册或写完重要日记之后。
  4. 图片越多，数据包越大；超大图片会提示并跳过，建议定期整理相册。
